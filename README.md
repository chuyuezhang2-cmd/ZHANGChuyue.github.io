## 📌 About This Project

This is a **professional job-seeking personal website** built for my application as **Data Analyst / Business Intelligence Intern**.

Designed and developed from scratch using modern web technologies, the site showcases my academic background, technical skills, four in-depth data/AI projects, and contact information. It is fully responsive, clean, and optimized for both desktop and mobile viewing.

The website strictly follows the course requirements of the **Final Individual Project — Personal Job-Seeking Website** and is deployed on **GitHub Pages**.
